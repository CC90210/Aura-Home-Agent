# AURA Product List

Reference hardware list for a standard AURA apartment install. Pricing varies by region and retailer, so treat these as planning categories rather than a fixed quote.

| Category | Recommended Item | Notes |
|---|---|---|
| Hub | Raspberry Pi 5 (8GB) | Runs Home Assistant OS and AURA services |
| Storage | 32GB+ microSD or USB SSD | SSD preferred for reliability |
| Microphone | USB microphone | Required for clap detection and voice agent |
| Primary lighting | Govee LED strips | Enable LAN Control in the Govee app |
| Smart plugs | TP-Link Kasa EP25 | Works well for appliance automation |
| Speaker | Sonos or HA-compatible speaker | Used for music and spoken responses |
| Lock | Home Assistant-compatible smart lock | Sensitive voice actions require a PIN |
| Cameras | Home Assistant-compatible cameras | Keep them on local network when possible |
| Tablet | Wall-mounted dashboard tablet | Optional dashboard surface |

For wiring, placement, and installation guidance, pair this with [HARDWARE_SETUP.md](./HARDWARE_SETUP.md).
